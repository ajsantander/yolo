A simple cli tool for farmers trying to make an honest living.

### Installation:
```
npm install --global yolo-farm
```

### Usage:
```
yolo <command> [options]
```

### Documentation:
```
yolo --help
```
or, for command specific documentation.
```
yolo <command> --help
```
